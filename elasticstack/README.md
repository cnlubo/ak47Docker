# 运行

```bash
# 启动
docker-compose up

# 启动 后台运行
docker-compose up -d

# 停止容器
docker-compose down

# 停止容器并且移除数据
docker-compose down -v

# 访问地址
# elasticsearch
http://127.0.0.1:9200/

# kibana
http://localhost:5601/

#  cerebro
http://localhost:9000/


```
